
# 使用说明

### ⚠️ 重要提示

> **注意！** 请务必保持此插件更新为打开状态。当游戏本体更新后，插件也需要同步更新语言文件，否则可能会导致游戏内新增的道具、NPC 或任务等内容无法正常显示文本（显示为空白）。

### 📝 操作步骤

1.  在游戏内按 `Enter` 键打开聊天框，切换到 **Toolbox 代理频道** (默认为 `/8` 或 `/proxy`)。

2.  输入以下命令并按 `Enter` 发送：
    ```
    lang cn
    ```

3.  执行命令后，**游戏客户端会立即关闭或闪退，这是正常现象**。

4.  手动重新启动游戏，您会发现界面和数据已切换为简体中文。

---

### 附：切换回英文

如果您需要将游戏数据切换回英文（保留中文UI和字体），可以在 Toolbox 代理频道输入：
```
lang en
```

同样，此操作也立即关闭游戏才能完全生效。
