
# language-pack-cn

这是一个Arborea Reborn中文语言支持插件，提供简体中文/英文的切换功能。

---

## 使用说明

#### 1、开启插件更新 (重要)

> **注意！** 请务必保持此插件为最新版本。当游戏本体更新后，插件也需要同步更新语言文件，否则可能会导致游戏内新增的道具、NPC 或任务等内容无法正常显示文本（显示为空白）。

---

####  2、检查fonts_tw.gpk文件

在执行切换中文的命令之前，**必须先进行以下检查和操作**，否则会导致插件报错！

1.  打开您的 TERA 游戏安装目录，找到以下路径：
    `TERA\S1Game\CookedPC\Art_Data\Packages\S1UI`

2.  检查该文件夹内是否存在名为 `fonts_tw.gpk` 的文件。

3.  **如果存在此文件，请务必将其删除。** 如果不存在，则无需任何操作。

4.  完成此项检查后，再进入游戏执行后续命令。

---
## 使用命令

#### 1、切换至简体中文

完成上述准备工作后，请按以下步骤操作：

1.  在游戏内按 `Enter` 键打开聊天框，切换到 **Toolbox 代理频道** (默认为 `/8` 或 `/proxy`)。

2.  输入以下命令并按 `Enter` 发送：
    ```
    lang cn
    ```

3.  执行命令后，**游戏客户端会立即关闭或闪退，这是正常现象**。

4.  手动重新启动游戏，您会发现界面和数据已切换为简体中文。

---

#### 2、切换回英文

如果您需要将游戏数据切换回英文（UI和字体仍可保留中文），可以在 Toolbox 代理频道输入：
```
lang en
```
同样，此操作也需要立即关闭游戏才能完全生效。
